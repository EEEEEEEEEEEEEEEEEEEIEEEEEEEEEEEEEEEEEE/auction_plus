## 简要说明
* AuctionPlus是拍卖行增加插件，核心功能一键购买方便扫货。
* 其他拍卖行插件扫描过程非常慢，AuctionPlus没有扫描过程，在拍卖行窗口增加一个勾选框，启用后，点击商品则为一口价一键购买。
* AuctionPlus使用方法简单，将AuctionPlus文件夹放到Interface/AddOns下，插件列表启用AuctionPlus。
